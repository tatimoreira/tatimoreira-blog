# Hello Rust

This is a test post to verify the Tera template pipeline works end to end.

## Why Rust?

Rust gives you:

- **Memory safety** without a garbage collector
- **Zero-cost abstractions** — high level code, low level performance
- **Fearless concurrency** — the compiler catches data races at compile time

## A simple example

```rust
fn main() {
    let message = "Hello, world!";
    println!("{}", message);
}
```

## Conclusion

If the template rendered this correctly, you should see proper headings, a list, and a code block above.
